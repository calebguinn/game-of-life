Conway's Game of Life in Javacript

This is an implementation of Conway's Game of Life or more precisely, the Hashlife-algorithm, written in Javascript using the canvas-tag.

It's probably the most feature-rich Game of Life that ever ran in your browser, featuring an unbound universe, a super-fast step mode and a whole bunch of examples. You can import patterns, change the interface, set custom rules, etc.

Technical details or the source may be published in the future.
